## 3. 分析 LED_cmake 工程

**用户:** 分析第二个文件夹内容

**回复:** 这是一个 **STM32F103C8T6 嵌入式项目**，由 STM32CubeMX 生成，使用 CMake + GCC 构建。功能是 LED 闪烁（PC13 引脚每秒翻转一次）。

项目结构：
```
LED_cmake/
├── Core/                        # 用户代码
│   ├── Inc/                     # 头文件 (gpio.h, main.h, ...)
│   └── Src/                     # 源文件 (main.c, gpio.c, ...)
├── Drivers/                     # HAL 驱动库 + CMSIS
├── cmake/stm32cubemx/           # CubeMX CMake 子项目
├── build/Debug/                 # 构建输出
├── CMakeLists.txt               # 顶层 CMake 脚本
├── LED_cmake.ioc                # CubeMX 配置文件
└── startup_stm32f103xb.s        # 启动汇编
```

关键参数：
- MCU: STM32F103C8T6 (Cortex-M3)
- 主频: 72MHz (HSE 8MHz → PLL ×9)
- HAL 库: STM32Cube FW_F1 V1.8.7

## 4. 查看主程序

**用户:** 主程序是什么

**回复:** `Core/Src/main.c`，核心逻辑：
```c
int main(void) {
    HAL_Init();                  // 初始化 HAL 库
    SystemClock_Config();        // 配置时钟 72MHz
    MX_GPIO_Init();              // 初始化 GPIO (PC13 推挽输出)
    while (1) {
        HAL_GPIO_TogglePin(GPIOC, GPIO_PIN_13);  // 翻转 PC13
        HAL_Delay(1000);                          // 延时 1 秒
    }
}
```

本质是 STM32 的 "Hello World" — 板载 LED 以 1Hz 频率闪烁。
